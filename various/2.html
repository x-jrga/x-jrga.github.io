Using Firewall Rules To Protect Computer Against Malicious Activity On Old Windows 7 Machine

These are my simple firewall rules. All suggestions and comments are welcome.

1. Go to Start Menu, search for wf.msc (Windows Firewall Advanced Security MMC Snap-in) and select it.

2. On right hand side, click on import policy. Select policy (windows7_firewall_rules.wfw) and open it.

You should see a message like this.

3. Click on Properties to verify firewall state.

4. Inbound rules are empty.

5. Outbound rules show a few applications that have access to the internet.

6. Modify to suit your needs.
