Testing Look and Feel Themes On The Swing Set Demo

My objective is a new practical and pleasing look for my applications.

I need a new java look and feel theme for my applications so I chose The Swingset2 Demo as the application where I would test the new themes in. I am creating a few themes to get a feel for the effect of the graphical user interface color on the user and to familiarize myself with the theme design process. Here are a few examples. Text should be easy to see and color combination should cause no eye strain.
