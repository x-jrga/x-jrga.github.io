PotatoSql: Learning Software For Database Design

PotatoSql allows me to create database schemas with minimal effort and to quickly generate corresponding sql source code for Hsqldb including referential integrity rules.

PotatoSql
  • allows me to design database schemas
  • generates ddl and dml code
  • allows me to build prototypes faster and less expensively
  • can be used as a learning/teaching tool
  • is free and open source software
Installation
  1. unzip PotatoSQL-30.zip
  2. cd PotatoSQL-30
  3. java -jar PotatoSQL-30.jar (if you are on windows double click on jar file)
